# PWA应用离线同步使用说明

## 项目概述

本项目是一个基于PWA技术的离线优先笔记和相册应用，支持多设备数据同步功能。用户可以在不同设备上离线记录笔记和上传相册，当服务器可用时自动同步数据，实现多设备间的数据共享。

### 核心功能

- ✅ 完全支持离线使用
- ✅ 自动数据同步
- ✅ 多设备数据共享
- ✅ 笔记记录和管理
- ✅ 相册上传和管理
- ✅ 零成本部署
- ✅ 数据自托管

## 系统要求

### 服务器端（用户电脑）

- Windows 10/11系统
- Node.js 16+（[下载链接](https://nodejs.org/)）
- npm 6+（Node.js内置）
- Ngrok（[下载链接](https://ngrok.com/download)）

### 客户端（手机）

- 支持PWA的现代浏览器（Chrome、Firefox、Safari等）
- 网络连接（用于同步数据）

## 安装和配置

### 1. 服务器端设置

#### 1.1 下载项目文件

将项目文件下载到电脑的任意目录，例如：`C:\Users\用户名\Documents\PWA-Sync-App`

#### 1.2 安装依赖

1. 打开命令提示符（CMD）或PowerShell
2. 导航到项目目录
3. 运行以下命令安装依赖：

```bash
npm install
```

#### 1.3 启动服务器

有两种启动方式：

##### 方式一：手动启动

1. 启动本地服务器：
   ```bash
   node server.js
   ```

2. 启动Ngrok进行内网穿透：
   ```bash
   ngrok http 3000
   ```

##### 方式二：使用启动脚本（推荐）

双击运行项目中的 `start-server.cmd` 文件，脚本会自动：
- 检查环境
- 安装依赖
- 启动本地服务器
- 启动Ngrok进行内网穿透

### 2. PWA应用配置

#### 2.1 安装PWA应用

1. 在手机浏览器中访问GitHub Pages部署的应用地址
2. 点击浏览器菜单中的「添加到主屏幕」或「安装应用」
3. 按照提示完成安装

#### 2.2 配置服务器地址

1. 打开安装好的PWA应用
2. 点击右上角的「登录设置」按钮
3. 在「服务器配置」部分，输入Ngrok生成的HTTPS URL
4. 点击「保存服务器地址」按钮
5. 应用将自动尝试同步数据

## 使用步骤

### 1. 日常使用

1. **离线记录笔记**：无需网络连接，直接在应用中添加笔记
2. **上传相册**：从手机相册选择照片上传
3. **查看内容**：随时查看已保存的笔记和相册
4. **自动同步**：当检测到服务器可用时，应用将自动同步数据

### 2. 数据同步流程

1. **设备A**：离线记录笔记和上传相册
2. **设备B**：离线记录笔记和上传相册
3. **晚上**：用户启动电脑，运行服务器和Ngrok
4. **设备A/B**：打开应用，自动检测到服务器可用，开始同步数据
5. **同步完成**：所有设备的数据保持一致

## 常见问题和解决方案

### 1. 服务器启动失败

**问题**：运行 `node server.js` 时出现错误

**解决方案**：
- 确保已安装Node.js和npm
- 确保已运行 `npm install` 安装依赖
- 检查端口3000是否被占用

### 2. Ngrok无法启动

**问题**：运行 `ngrok http 3000` 时出现错误

**解决方案**：
- 确保已正确安装Ngrok
- 检查Ngrok是否已登录（首次使用需要登录）
- 尝试重启Ngrok

### 3. 应用无法同步数据

**问题**：应用显示「无法连接到服务器」

**解决方案**：
- 检查服务器是否正在运行
- 检查Ngrok是否正在运行
- 检查服务器URL是否正确
- 检查手机网络连接

### 4. 数据冲突

**问题**：多设备同时修改同一条数据

**解决方案**：
- 系统会自动按时间戳解决冲突，保留最新的数据
- 建议尽量避免同时修改同一条数据

## 技术说明

### 数据存储

- **客户端**：数据存储在浏览器的LocalStorage中
- **服务器端**：数据存储在电脑本地的JSON文件中（`sync-data.json`）

### 同步机制

1. **待同步队列**：所有操作都添加到待同步队列
2. **定时检查**：应用每分钟检查一次服务器可用性
3. **网络检测**：当检测到网络连接和服务器可用时，开始同步
4. **数据合并**：按时间戳合并数据，解决冲突

### 安全说明

- 数据传输使用HTTPS加密
- 所有数据存储在用户本地，隐私安全
- 无第三方数据收集
